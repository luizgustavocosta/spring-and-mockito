# spring-and-mockito
Projeto para demostrar boas práticas no uso do Mockito com Spring

## Construído com
* Java 21
* Spring Boot 3.3.2
* H2 Database
* RabbitMQ
* Codium
* Refact.ai
* Cursor.ai

## Como executar
* mvn clean verify
* mvn spring-boot:run
